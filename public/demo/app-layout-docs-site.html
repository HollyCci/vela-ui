Introduction

HeroUI is a modern React component library built on top of React Aria Components and Tailwind CSS v4. It provides accessible, customizable UI primitives for building high-quality web applications.

Features

  • Accessible by default with React Aria
  • Tailwind CSS v4 with design tokens
  • Compound component patterns
  • Dark mode support
  • Server Component compatible
